Functionality

## Folder Best Model 1 -- generate the submission result 1 - please set working directory to XX/Group_09/Code/Best Model 1 before run any code in the folder

"CART.r" -- used to create two CART models with different prediction results and return prediction results
"randomForest.r" -- used to create a randomForest model and return prediction result
